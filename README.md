# Commonly used commands (leader = space):
space + space: fuzzy file name search
space + /: fuzzy grep
space + .: scratch pad for notes
space + ,: switch between open buffers
shift + h|l: cycle between buffers
g + r: see all references
g + d: see definition
space + c + i|o: show call tree in|out
ctrl + o: jump back to last location in code
space + g + b: git blame (very useful to find the grizzly url)
